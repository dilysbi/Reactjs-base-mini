## Clone

```
git clone https://github.com/dilysngo/Reactjs-BASE-mini.git

with yarn:

yarn install
yarn start

with npm:

npm install
npm start
```

## Setup

```
.env.development for development
.env.staging for testing staging
.env.production for build production
```
